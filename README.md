# 🧠 Maze BFS Visualizer

## 📌 Описание
Программа для поиска выхода из лабиринта с использованием алгоритма BFS (поиск в ширину) с графическим интерфейсом.

## 🚀 Возможности
- Редактор лабиринта (можно рисовать стены)
- Алгоритм BFS для поиска кратчайшего пути
- Визуализация пути
- Анимация движения
- Простой интерфейс (Tkinter)

## 🧠 Алгоритм
Используется алгоритм Breadth-First Search (BFS), который гарантирует нахождение кратчайшего пути.
